# تطبيق خواطر إيمانية

تطبيق خواطر إيمانية هو تطبيق بسيط يعمل باستخدام إطار عمل Flutter و BLoC، ويهدف إلى عرض نصوص دينية إيمانية تعزز الروحانية والتأمل. يتميز التطبيق بواجهة سهلة الاستخدام وتجربة مريحة لعرض المحتوى.

## الميزات

- **عرض الخواطر الإيمانية**: يتم تقسيم النصوص إلى صفحات تسهّل قراءة المحتوى والتنقل بين الأفكار
- **استخدام BLoC**: لإدارة الحالة وتحسين الأداء، مما يضمن سلاسة التطبيق وكفاءة في عرض البيانات
- **تنقل سهل**: التنقل بين الصفحات بشكل أفقي لعرض المحتوى حسب حجم الشاشة
- **إشعارات يومية**: جدولة 5 إشعارات يومية لتذكير المستخدم بالخواطر الإيمانية
- **دعم النصوص العربية**: التطبيق يدعم اللغة العربية بشكل كامل لتوفير تجربة سلسة ومناسبة للمستخدمين
- **تخصيص حجم الخط**: إمكانية تكبير وتصغير حجم النصوص حسب الرغبة
